Class for feature detection inside images and videos with the RANSAC algorithm.

feature¶

tra.features.Feature

the feature class.

max_it¶

int

Max Number of iterations for the RANSAC loop.

inliers_percent¶

float

percentage of inliers over total points.Used for stop criteria in the RANSAC loop.

threshold¶

int

Value of threshold used after the image normalization (can be an integer from 1 to 254).

dst¶

float

the distance of the inliers pixels from the feature (i.e. a pixel is considered an inlier if its distance is < dst).
